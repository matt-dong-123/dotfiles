- tmux
- zshrc

## How to apply settings

1. Install [brew](https://brew.sh)
2. Install gnu stow (`brew install stow`)
3. Clone this repo (`git clone https://github.com/matt-dong-123/dotfiles.git`)
4. Run `stow .` in the dotfiles directory
